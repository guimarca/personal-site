## Getting Started

Install pnpm:

```bash
npm i -g pnpm
```

Run dev server:

```bash
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) in your browser.

```bash
npm run build
```

## Tests

```bash
npm run test:e2e
```

## Update dependencies

Install (if you have not not already)

```bash
npm i -g npm-check-updates
```

Then:

```bash
ncu --target semver -u
pnpm i
npm update
```
